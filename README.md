# redux starter


## How to use it
* Development (view the tool)
    * run
        1. `npm i`
        2. `npm run dev`
        3. browse `localhost:3001`
    * test
        1. `npm i`
        2. `npm run test`

* Production (build)
        1. `npm i`
        2. `npm run build`
        3. `npm run start`
